线性搜索

线性搜索（Linear Search）是最基础的搜索算法，也称为顺序搜索。它通过逐个检查数组或列表中的元素来查找目标值， 是信息学奥赛中必须掌握的基础算法之一。

线性搜索概述

什么是线性搜索？

线性搜索是一种简单的搜索算法，它从数据结构的第一个元素开始，逐个检查每个元素，直到找到目标值或搜索完所有元素。 线性搜索不需要数据结构是有序的，适用于任何类型的序列。

线性搜索的特点：

  • 实现简单，易于理解和编码
  • 不需要数据是有序的
  • 适用于小型数据集
  • 时间复杂度为 O(n)，n 是元素数量
  • 空间复杂度为 O(1)，不需要额外空间

线性搜索的基本步骤

1 从数组的第一个元素开始
2 将当前元素与目标值进行比较
3 如果找到匹配的元素，返回其索引位置
4 如果没有找到匹配的元素，继续检查下一个元素
5 如果搜索完所有元素仍未找到，返回一个表示未找到的值（通常是 -1）

#include <iostream>
using namespace std;
 
// 线性搜索函数
int linearSearch(int arr[], int size, int target) {
  for (int i = 0; i < size; i++) {
    if (arr[i] == target) {
      return i; // 找到目标，返回索引
    }
  }
  return -1; // 未找到目标
}
 
int main() {
  int arr[] = {12, 5, 9, 3, 17, 8, 21, 4};
  int size = sizeof(arr) / sizeof(arr[0]);
  int target = 17;
  
  int result = linearSearch(arr, size, target);
  
  if (result != -1) {
    cout << "元素找到，索引为: " << result << endl;
  } else {
    cout << "元素未找到" << endl;
  }
  
  return 0;
}

线性搜索代码示例 (C++)

1. 基础实现 - 查找单个元素

基本的线性搜索实现，查找数组中是否存在目标元素并返回其索引 

#include 
using namespace std;

// 线性搜索函数
// 参数: arr - 待搜索的数组
//       size - 数组大小
//       target - 目标值
// 返回值: 找到则返回索引，否则返回-1
int linearSearch(int arr[], int size, int target) {
    for (int i = 0; i < size; i++) {
        // 比较当前元素与目标值
        if (arr[i] == target) {
            return i; // 找到目标，返回索引
        }
    }
    return -1; // 未找到目标
}

int main() {
    int numbers[] = {5, 2, 9, 1, 5, 6};
    int size = sizeof(numbers) / sizeof(numbers[0]);
    int target = 9;
    
    int index = linearSearch(numbers, size, target);
    
    if (index != -1) {
        cout << "元素 " << target << " 找到，索引为: " << index << endl;
    } else {
        cout << "元素 " << target << " 未找到" << endl;
    }
    
    return 0;
}

2. 查找所有出现 - 多个匹配项

查找目标元素在数组中所有出现的位置，并存储这些索引 

#include 
#include 
using namespace std;

// 查找目标元素所有出现的位置
vector findAllOccurrences(int arr[], int size, int target) {
    vector indices; // 存储所有找到的索引
    
    for (int i = 0; i < size; i++) {
        if (arr[i] == target) {
            indices.push_back(i); // 将找到的索引添加到向量中
        }
    }
    
    return indices;
}

int main() {
    int numbers[] = {5, 2, 5, 1, 5, 6};
    int size = sizeof(numbers) / sizeof(numbers[0]);
    int target = 5;
    
    vector results = findAllOccurrences(numbers, size, target);
    
    if (results.empty()) {
        cout << "元素 " << target << " 未找到" << endl;
    } else {
        cout << "元素 " << target << " 出现的索引为: ";
        for (int i = 0; i < results.size(); i++) {
            cout << results[i];
            if (i < results.size() - 1) {
                cout << ", ";
            }
        }
        cout << endl;
    }
    
    return 0;
}

3. 应用示例 - 查找最大值

利用线性搜索的思想查找数组中的最大值 

#include 
using namespace std;

// 查找数组中的最大值
int findMax(int arr[], int size) {
    if (size <= 0) {
        return -1; // 处理空数组情况
    }
    
    int maxVal = arr[0]; // 假设第一个元素是最大值
    int maxIndex = 0;    // 记录最大值的索引
    
    // 线性搜索整个数组
    for (int i = 1; i < size; i++) {
        if (arr[i] > maxVal) {
            maxVal = arr[i];  // 更新最大值
            maxIndex = i;     // 更新最大值索引
        }
    }
    
    return maxIndex; // 返回最大值的索引
}

int main() {
    int numbers[] = {5, 2, 9, 1, 5, 6};
    int size = sizeof(numbers) / sizeof(numbers[0]);
    
    int maxIndex = findMax(numbers, size);
    
    if (maxIndex != -1) {
        cout << "数组中的最大值是: " << numbers[maxIndex] 
             << ", 位于索引 " << maxIndex << endl;
    } else {
        cout << "数组为空" << endl;
    }
    
    return 0;
}

时间复杂度分析

线性搜索的效率分析

线性搜索的时间复杂度为 O(n)，其中 n 是数组中的元素数量。这是因为在最坏情况下，我们可能需要检查数组中的所有元素。

最佳情况

目标元素是数组的第一个元素，只需要一次比较。
时间复杂度: O(1)

平均情况

目标元素位于数组的中间位置，需要检查大约 n/2 个元素。
时间复杂度: O(n)

最坏情况

目标元素是数组的最后一个元素或不存在，需要检查所有 n 个元素。
时间复杂度: O(n)

空间复杂度

线性搜索的空间复杂度为 O(1)，因为它不需要额外的存储空间，只使用固定数量的变量。

线性搜索的适用场景

  • 小型数据集，元素数量较少
  • 无序的数据集合
  • 需要查找多个匹配项的场景
  • 数据结构不支持随机访问（如链表）
  • 一次性搜索，不需要多次查询

注意：对于大型有序数据集，二分搜索（O(log n)）等算法通常比线性搜索更高效。但线性搜索实现简单，在许多竞赛场景中仍有广泛应用。
