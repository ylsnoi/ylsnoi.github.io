队列

队列是一种先进先出(FIFO)的线性数据结构，广泛应用于算法和程序设计中。本页面将详细讲解C++中队列的概念、实现、操作以及在信息学奥赛中的典型应用。

队列概念概述

什么是队列？

队列(Queue)是一种特殊的线性表，它只允许在表的前端（front）进行删除操作，而在表的后端（rear）进行插入操作。 进行插入操作的端称为队尾，进行删除操作的端称为队头。队列遵循"先进先出"（First In First Out, FIFO）的原则。

队列的基本操作

入队 (Enqueue)

在队列的尾部添加一个新元素。如果队列已满，则称为"上溢"（Overflow）。

出队 (Dequeue)

从队列的头部移除并返回一个元素。如果队列为空，则称为"下溢"（Underflow）。

查看队头 (Front)

返回队列头部的元素，但不将其从队列中移除。

判空 (Empty)

检查队列是否为空，如果为空则返回true，否则返回false。

C++ STL中的队列

C++标准模板库(STL)提供了队列的实现，位于<queue>头文件中，主要操作如下： 

#include <queue>  // 包含队列头文件
using namespace std;

// 声明一个队列
queue<int> q;  // 存储int类型的队列

// 入队操作
q.push(10);    // 向队列尾部添加元素10

// 出队操作
q.pop();       // 移除队列头部的元素

// 获取队头元素
int front = q.front();  // 返回队列头部的元素

// 判断队列是否为空
bool isEmpty = q.empty();  // 队列为空返回true，否则返回false

// 获取队列大小
int size = q.size();  // 返回队列中元素的个数

队列的两种主要实现方式

数组实现（循环队列）

// 循环队列的数组实现
const int MAX_SIZE = 100;
int queue[MAX_SIZE];
int front = 0;  // 队头指针
int rear = 0;   // 队尾指针
int count = 0;  // 队列元素数量

// 入队操作
bool enqueue(int value) {
    if (count == MAX_SIZE) return false;  // 队列已满
    queue[rear] = value;
    rear = (rear + 1) % MAX_SIZE;  // 循环移动队尾指针
    count++;
    return true;
}

// 出队操作
bool dequeue() {
    if (count == 0) return false;  // 队列为空
    front = (front + 1) % MAX_SIZE;  // 循环移动队头指针
    count--;
    return true;
}

// 获取队头元素
int getFront() {
    if (count == 0) return -1;  // 队列为空
    return queue[front];
}

数组实现的队列通常设计为循环队列，以充分利用存储空间，避免"假溢出"问题。

链表实现

// 链表节点结构
struct Node {
    int data;
    Node* next;
    Node(int val) : data(val), next(nullptr) {}
};

// 链表队列
class Queue {
private:
    Node* front;  // 队头指针
    Node* rear;   // 队尾指针
    int count;    // 元素数量

public:
    Queue() : front(nullptr), rear(nullptr), count(0) {}
    
    // 入队操作
    void enqueue(int val) {
        Node* newNode = new Node(val);
        if (isEmpty()) {
            front = rear = newNode;
        } else {
            rear->next = newNode;
            rear = newNode;
        }
        count++;
    }
    
    // 出队操作
    bool dequeue() {
        if (isEmpty()) return false;
        Node* temp = front;
        front = front->next;
        delete temp;
        count--;
        if (front == nullptr) rear = nullptr;
        return true;
    }
    
    // 获取队头元素
    int getFront() {
        if (isEmpty()) return -1;
        return front->data;
    }
    
    // 判断是否为空
    bool isEmpty() {
        return count == 0;
    }
};

链表实现的队列可以动态调整大小，不会出现空间不足的问题，但需要额外的指针存储空间。

队列的算法应用

队列在信息学奥赛中的典型应用

广度优先搜索(BFS)

队列是实现广度优先搜索的核心数据结构，用于按层次遍历图或树，如最短路径问题、迷宫求解等。

模拟排队问题

处理各类排队模拟问题，如银行排队、超市结账等，计算平均等待时间等统计信息。

缓冲区设计

在处理输入输出、任务调度等场景中用作缓冲区，平衡数据处理速度差异。

单调队列

一种特殊的队列，用于高效解决滑动窗口最值等问题，是竞赛中的高级技巧。

BFS算法中的队列应用示例

下面是使用队列实现广度优先搜索，求解迷宫最短路径的示例代码： 

#include <iostream>
#include <queue>
#include <vector>
using namespace std;

// 迷宫大小
const int MAX_N = 100;
const int MAX_M = 100;

// 迷宫地图
int maze[MAX_N][MAX_M];
// 记录步数
int steps[MAX_N][MAX_M];
// 方向数组：上下左右
int dx[] = {-1, 1, 0, 0};
int dy[] = {0, 0, -1, 1};

int n, m;  // 迷宫的行数和列数
int start_x, start_y;  // 起点坐标
int end_x, end_y;      // 终点坐标

// BFS求解最短路径
int bfs() {
    // 初始化步数数组为-1（未访问）
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < m; j++) {
            steps[i][j] = -1;
        }
    }
    
    // 创建队列，存储坐标
    queue<pair<int, int>> q;
    
    // 起点入队
    q.push({start_x, start_y});
    steps[start_x][start_y] = 0;
    
    // 队列不为空时循环
    while (!q.empty()) {
        // 取出队头元素
        pair<int, int> current = q.front();
        q.pop();
        int x = current.first;
        int y = current.second;
        
        // 如果到达终点，返回步数
        if (x == end_x && y == end_y) {
            return steps[x][y];
        }
        
        // 探索四个方向
        for (int i = 0; i < 4; i++) {
            int nx = x + dx[i];
            int ny = y + dy[i];
            
            // 判断新坐标是否合法且未访问过且不是障碍物
            if (nx >= 0 && nx < n && ny >= 0 && ny < m 
                && steps[nx][ny] == -1 && maze[nx][ny] == 0) {
                // 新坐标入队
                q.push({nx, ny});
                // 步数加1
                steps[nx][ny] = steps[x][y] + 1;
            }
        }
    }
    
    // 无法到达终点
    return -1;
}

int main() {
    cin >> n >> m;
    
    // 读取迷宫
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < m; j++) {
            cin >> maze[i][j];
        }
    }
    
    // 读取起点和终点
    cin >> start_x >> start_y >> end_x >> end_y;
    
    // 求解并输出结果
    int result = bfs();
    cout << "最短路径长度: " << result << endl;
    
    return 0;
}

在BFS算法中，队列用于存储待访问的节点，确保按照距离起点由近及远的顺序进行访问，从而找到最短路径。
