字符串概述

什么是字符串？

字符串是由零个或多个字符组成的有限序列，是信息学奥赛中处理文本数据的基本方式。在C++中，有两种主要的字符串表示方式： C风格字符串（字符数组）和C++标准库的string类。

信奥中字符串的应用场景：

  • 文本处理与分析
  • 字符串匹配算法（如KMP算法）
  • 密码学相关问题（加密、解密）
  • 模拟问题中的文本输出
  • 格式化数据处理

C风格字符串

本质是字符数组，以'\0'作为终止符。需要手动管理内存，容易出现越界问题，但执行效率高，适合对性能要求高的场景。

标准库提供的字符串类，封装了字符串操作，自动管理内存，支持动态长度，是信奥中推荐使用的字符串类型。

// 方法1: 使用库函数
string reverseString1(string s) {
    reverse(s.begin(), s.end());
    return s;
}

// 方法2: 手动交换
string reverseString2(string s) {
    int n = s.size();
    for (int i = 0; i < n / 2; i++) {
        swap(s[i], s[n - 1 - i]);
    }
    return s;
}

int main() {

判断回文串

回文串是指正读和反读都相同的字符串，是信奥常见问题。 

#include 
#include 
#include 
using namespace std;

// 判断是否为回文串（忽略大小写和非字母字符）
bool isPalindrome(string s) {
    int left = 0, right = s.size() - 1;
    
    while (left < right) {
        // 跳过非字母字符
        while (left < right && !isalpha(s[left])) left++;
        while (left < right && !isalpha(s[right])) right--;
        
        // 转换为小写比较
        if (tolower(s[left]) != tolower(s[right])) {
            return false;
        }
        
        left++;
        right--;
    }
    return true;
}

int main() {
    cout << isPalindrome("A man a plan a canal Panama") << endl; // 1
    cout << isPalindrome("hello") << endl; // 0
    return 0;
}

字符串匹配（暴力算法）

在主串中查找子串的位置，是字符串处理的核心问题之一。 

#include 
#include 
using namespace std;

// 暴力匹配算法
int findSubstring(string mainStr, string subStr) {
    int n = mainStr.size();
    int m = subStr.size();
    
    // 主串剩余长度不足子串长度，无需比较
    for (int i = 0; i <= n - m; i++) {
        int j;
        // 比较子串与主串中从i开始的字符
        for (j = 0; j < m; j++) {
            if (mainStr[i + j] != subStr[j]) {
                break;
            }
        }
        // 找到匹配的子串
        if (j == m) {
            return i; // 返回起始位置
        }
    }
    return -1; // 未找到
}

int main() {
    string s = "abcdeabcdf";
    string sub = "abcd";
    cout << findSubstring(s, sub) << endl; // 输出 0
    return 0;
}

KMP算法（高效字符串匹配）

KMP算法是一种高效的字符串匹配算法，通过预处理子串构建部分匹配表，避免不必要的字符比较，时间复杂度为O(n+m)。 

#include 
#include 
#include 
using namespace std;

// 构建部分匹配表(前缀函数)
vector computeLPS(string pattern) {
    int m = pattern.size();
    vector lps(m, 0);
    int len = 0; // 最长前缀后缀的长度
    
    for (int i = 1; i < m; ) {
        if (pattern[i] == pattern[len]) {
            len++;
            lps[i] = len;
            i++;
        } else {
            if (len != 0) {
                len = lps[len - 1];
            } else {
                lps[i] = 0;
                i++;
            }
        }
    }
    return lps;
}

// KMP匹配算法
int kmpSearch(string text, string pattern) {
    int n = text.size();
    int m = pattern.size();
    
    if (m == 0) return 0; // 空模式串
    
    vector lps = computeLPS(pattern);
    int i = 0; // 文本索引
    int j = 0; // 模式串索引
    
    while (i < n) {
        if (pattern[j] == text[i]) {
            i++;
            j++;
        }
        
        if (j == m) {
            return i - j; // 找到匹配，返回起始位置
            j = lps[j - 1];
        } else if (i < n && pattern[j] != text[i]) {
            if (j != 0) {
                j = lps[j - 1];
            } else {
                i++;
            }
        }
    }
    return -1; // 未找到
}

int main() {
    string text = "ABABDABACDABABCABAB";
    string pattern = "ABABCABAB";
    cout << "模式串在文本中的位置: " << kmpSearch(text, pattern) << endl; // 10
    return 0;
}

例题1: 统计字符出现次数

问题描述: 输入一个字符串，统计其中每个字符出现的次数，并按字符顺序输出。 

#include 
#include 
using namespace std;

int main() {
    string s;
    getline(cin, s); // 读取整行字符串，包括空格
    
    // 统计每个字符出现的次数
    int count[256] = {0}; // 假设使用ASCII字符集
    for (char c : s) {
        count[(unsigned char)c]++;
    }
    
    // 输出统计结果
    for (int i = 0; i < 256; i++) {
        if (count[i] > 0) {
            cout << "'" << (char)i << "': " << count[i] << "次" << endl;
        }
    }
    
    return 0;
}

解题思路:

  • 使用getline()函数读取完整输入，包括空格
  • 创建大小为256的数组（覆盖所有ASCII字符）存储每个字符的计数
  • 遍历字符串，对每个字符进行计数
  • 遍历计数数组，输出所有出现过的字符及其计数

例题2: 字符串加密

问题描述: 实现一个简单的凯撒密码加密算法，将字符串中的每个字母向后移动k位（循环移动，z后是a），非字母字符不变。 

#include 
#include 
using namespace std;

// 凯撒密码加密
string caesarEncrypt(string s, int k) {
    k %= 26; // 确保k在0-25之间
    if (k < 0) k += 26; // 处理负数情况
    
    for (int i = 0; i < s.size(); i++) {
        // 处理大写字母
        if (s[i] >= 'A' && s[i] <= 'Z') {
            s[i] = (s[i] - 'A' + k) % 26 + 'A';
        }
        // 处理小写字母
        else if (s[i] >= 'a' && s[i] <= 'z') {
            s[i] = (s[i] - 'a' + k) % 26 + 'a';
        }
        // 非字母字符不变
    }
    return s;
}

int main() {
    string s;
    int k;
    getline(cin, s);
    cin >> k;
    
    cout << "加密前: " << s << endl;
    cout << "加密后: " << caesarEncrypt(s, k) << endl;
    
    return 0;
}

解题思路:

  • 对移动位数k进行取模26操作，确保在合理范围内
  • 区分处理大写字母和小写字母，保持原大小写不变
  • 使用字符ASCII码值进行计算，实现循环移动效果
  • 非字母字符不进行处理，保持原样

例题3: 最长公共前缀

问题描述: 输入n个字符串，找出它们的最长公共前缀。如果不存在公共前缀，输出空字符串。 

#include 
#include 
#include 
using namespace std;

// 查找最长公共前缀
string longestCommonPrefix(vector& strs) {
    if (strs.empty()) return "";
    
    // 以第一个字符串为基准
    for (int i = 0; i < strs[0].size(); i++) {
        char c = strs[0][i];
        // 与其他字符串的对应位置比较
        for (int j = 1; j < strs.size(); j++) {
            // 如果到达某个字符串的末尾，或者字符不匹配
            if (i == strs[j].size() || strs[j][i] != c) {
                return strs[0].substr(0, i);
            }
        }
    }
    return strs[0]; // 所有字符串都相同
}

int main() {
    int n;
    cin >> n;
    cin.ignore(); // 忽略换行符
    
    vector strs(n);
    for (int i = 0; i < n; i++) {
        getline(cin, strs[i]);
    }
    
    cout << "最长公共前缀: " << longestCommonPrefix(strs) << endl;
    
    return 0;
}

解题思路:

  • 以第一个字符串为基准，逐个字符与其他字符串比较
  • 对于每个位置，检查所有字符串的对应位置是否有相同字符
  • 一旦发现不匹配或到达某个字符串的末尾，就返回当前找到的前缀
  • 如果所有字符都匹配，则第一个字符串就是最长公共前缀

常见错误与注意事项

字符串处理中的常见错误

数组越界访问

访问字符串时超出其实际长度范围，尤其是使用C风格字符串时，忘记终止符'\0'会导致此问题。

字符串比较错误

对C风格字符串使用"=="进行比较（实际比较的是地址而非内容），应使用strcmp函数。

输入处理不当

使用cin读取字符串时遇到空格会停止，需要读取整行时应使用getline函数。

忽略字符编码

处理中文等多字节字符时，误用单字节处理方式导致乱码或处理错误。

信奥中的字符串处理技巧

  • 预处理输入: 读取字符串后先进行清洗（如去除空格、转换大小写），统一格式
  • 善用标准库: 熟悉并使用string类的成员函数，避免重复造轮子
  • 注意效率: 频繁拼接字符串时，考虑使用reserve()预分配空间提高效率
  • 字符与ASCII码转换: 灵活运用字符与ASCII码的转换处理数字字符等
  • 边界情况处理: 特别注意空字符串、单字符字符串等边界情况
  • 算法选择: 小规模字符串匹配可用暴力算法，大规模数据应使用KMP等高效算法
