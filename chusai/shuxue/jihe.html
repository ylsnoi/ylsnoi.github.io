集合与逻辑

集合与逻辑是CSP初赛的基础知识点，包括集合运算、逻辑命题和布尔代数等内容， 是信息学奥赛中算法设计和问题分析的重要工具。

集合运算 逻辑命题

集合运算

集合的基本概念

集合是由一些确定的、不同的对象组成的整体。集合中的对象称为元素。在信息学中，集合常用于表示数据的分组和分类。

CSP初赛中常用的集合表示方法：

  • 列举法：将集合中的元素一一列举出来，如 {1, 2, 3, 4}
  • 描述法：用元素所具有的共同性质来表示集合，如 {x | x是偶数且x ≤ 10}
  • 区间表示法：常用于表示连续的整数集合，如 [1, 10] 表示1到10的所有整数

集合运算定律

交换律

A ∪ B = B ∪ A

A ∩ B = B ∩ A

结合律

(A ∪ B) ∪ C = A ∪ (B ∪ C)

(A ∩ B) ∩ C = A ∩ (B ∩ C)

分配律

A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)

A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)

德摩根律

¬(A ∪ B) = ¬A ∩ ¬B

¬(A ∩ B) = ¬A ∪ ¬B

集合相关例题

例题1：集合的基本运算

设集合A = {1, 2, 3, 4, 5}，集合B = {3, 4, 5, 6, 7}，则A ∪ B = ______，A ∩ B = ______，A - B = ______。

答案：

A ∪ B = {1, 2, 3, 4, 5, 6, 7}

A ∩ B = {3, 4, 5}

A - B = {1, 2}

例题2：集合的计数

一个集合有n个元素，那么它有______个子集，有______个真子集。

答案： 2ⁿ，2ⁿ - 1

解析： 每个元素都有两种选择：属于子集或不属于子集，所以总共有2ⁿ个子集。真子集是不包含集合本身的子集，所以有2ⁿ - 1个。

逻辑命题与布尔代数

逻辑命题的基本概念

逻辑命题是具有确定真假意义的陈述句。在信息学中，逻辑命题是条件判断、循环控制等程序结构的基础。

CSP初赛中常用的逻辑运算符：

否定 (¬)

非运算，如¬P表示"P不成立"

合取 (∧)

与运算，如P∧Q表示"P和Q都成立"

析取 (∨)

或运算，如P∨Q表示"P或Q成立"

蕴含 (→)

如P→Q表示"如果P成立，则Q成立"

等价 (↔)

如P↔Q表示"P成立当且仅当Q成立"

异或 (⊕)

如P⊕Q表示"P和Q中有且仅有一个成立"

运算规则说明

与运算 (P ∧ Q)：当且仅当P和Q都为真时，结果为真，否则为假。

在C++中，与运算用&&表示，如if (a > 0 && b < 10)

应用场景：需要同时满足多个条件时使用，如判断一个数是否在某个范围内。

C++中的短路特性

C++中的逻辑运算符具有短路特性：

  • 对于&&，如果左侧表达式为假，则右侧表达式不会被执行
  • 对于||，如果左侧表达式为真，则右侧表达式不会被执行
// 短路特性示例
int a = 0, b = 5;
if (a != 0 && ++b > 5) {  // a != 0为假，++b不会执行
    // 不会进入这里
}
cout << b;  // 输出5，而不是6

布尔代数定律

交换律

P ∧ Q = Q ∧ P

P ∨ Q = Q ∨ P

结合律

(P ∧ Q) ∧ R = P ∧ (Q ∧ R)

(P ∨ Q) ∨ R = P ∨ (Q ∨ R)

分配律

P ∧ (Q ∨ R) = (P ∧ Q) ∨ (P ∧ R)

P ∨ (Q ∧ R) = (P ∨ Q) ∧ (P ∨ R)

德摩根律

¬(P ∧ Q) = ¬P ∨ ¬Q

¬(P ∨ Q) = ¬P ∧ ¬Q

吸收律

P ∧ (P ∨ Q) = P

P ∨ (P ∧ Q) = P

双重否定律

¬(¬P) = P

P ∨ ¬P = 真 (排中律)

CSP初赛解题技巧

集合运算题解题技巧

  • 对于集合计数问题，牢记n个元素的集合有2ⁿ个子集
  • 利用文氏图(Venn Diagram)辅助分析多个集合间的关系
  • 涉及集合运算时，可先将集合转化为二进制表示（元素存在为1，否则为0），再进行位运算
  • 德摩根律在处理集合补集运算时非常有用：¬(A ∪ B) = ¬A ∩ ¬B

逻辑推理题解题技巧

  • 复杂逻辑表达式可通过真值表进行简化
  • 注意区分"或"运算与"异或"运算的区别
  • 蕴含运算P→Q等价于¬P∨Q，这是转换复杂逻辑的重要公式
  • C++中的逻辑运算符优先级：! > && > ||
  • 利用短路特性分析程序执行路径和变量变化

逻辑相关例题

例题1：逻辑表达式化简

逻辑表达式¬(P ∨ Q) ∨ (¬P ∧ Q)化简后等价于______。

答案： ¬P

解析：

根据德摩根律：¬(P ∨ Q) = ¬P ∧ ¬Q

原式 = (¬P ∧ ¬Q) ∨ (¬P ∧ Q) = ¬P ∧ (¬Q ∨ Q) = ¬P ∧ 真 = ¬P

例题2：程序逻辑分析

执行以下C++代码后，变量x的值是______。 

int a = 3, b = 5, x = 0;
if (a > 5 && (b = 10) > 0) {
    x = 1;
}
x = b;

答案： 5

解析： 由于a > 5为假，根据&&的短路特性，右侧的(b = 10)不会执行，b的值仍为5，因此x最终为5。

例题3：集合与逻辑综合题

设全集U = {1, 2, 3, 4, 5, 6, 7, 8, 9}，集合A = {x | x是偶数}，集合B = {x | x是3的倍数}， 则不属于A ∪ B的元素有______个。

答案： 3

解析：

A = {2, 4, 6, 8}，B = {3, 6, 9}

A ∪ B = {2, 3, 4, 6, 8, 9}

不属于A ∪ B的元素是{1, 5, 7}，共3个。
