线性结构

线性结构是CSP初赛的核心考点，包括数组、字符串、栈和队列。 本页面将系统讲解这些结构的概念、操作及在信息学奥赛中的应用。

数组 (Array)

数组的概念

数组是一种由相同类型元素组成的线性数据结构，元素在内存中连续存储，通过索引（下标）访问。 在C++中，数组的下标从0开始，这是CSP初赛的重要考点。

数组的特点

  • 元素类型相同，内存连续分配
  • 支持随机访问，访问时间复杂度为O(1)
  • 数组大小固定，定义后不能改变（动态数组除外）
  • 插入和删除元素效率低，时间复杂度为O(n)

C++数组定义与操作

// 1. 静态数组定义
int arr1[5];               // 定义包含5个int类型元素的数组
int arr2[5] = {1, 2, 3};   // 初始化数组，未赋值元素默认为0
int arr3[] = {1, 2, 3, 4}; // 自动推断数组大小为4

// 2. 访问数组元素
cout << arr2[0];           // 输出第一个元素（下标从0开始）
arr1[2] = 10;              // 给第三个元素赋值

// 3. 数组遍历
for (int i = 0; i < 5; i++) {
    cout << arr2[i] << " ";
}

// 4. 多维数组
int matrix[3][4];          // 3行4列的二维数组
matrix[1][2] = 5;          // 访问第二行第三列元素

字符串 (String)

字符串的概念

字符串是由字符组成的线性序列，在C++中可以用字符数组或string类表示。 字符串以空字符'\0'作为结束标志，这是CSP初赛的常考知识点。

字符串的特点

  • 由字符组成的有序序列
  • C风格字符串以'\0'结尾
  • string类自动管理内存，更安全易用
  • 支持拼接、比较、截取等操作

C++字符串操作

// 1. C风格字符串
char str1[10] = "hello";    // 字符数组，自动添加'\0'
char str2[] = "world";      // 自动计算长度

// 2. C++ string类
#include 
using namespace std;

string s1 = "hello";        // string对象
string s2("world");         // 另一种初始化方式

// 3. 字符串操作
cout << s1.length();        // 获取长度
cout << s1 + s2;            // 拼接字符串
cout << s1.substr(1, 3);    // 截取子串，从索引1开始，长度3
if (s1 == s2) { ... }       // 比较字符串

// 4. 字符串遍历
for (int i = 0; i < s1.length(); i++) {
    cout << s1[i];          // 访问单个字符
}

栈 (Stack)

栈的概念

栈是一种遵循"后进先出"(LIFO, Last In First Out)原则的线性结构。 只能在栈顶进行插入(push)和删除(pop)操作，这是CSP初赛的核心考点。

栈的特点

  • 后进先出(LIFO)的数据结构
  • 只能在栈顶进行操作
  • 插入操作称为入栈(push)
  • 删除操作称为出栈(pop)
  • 访问栈顶元素称为 peek 或 top

C++栈的实现与操作

#include   // 包含栈头文件
using namespace std;

// 1. 栈的定义
stack s;

// 2. 栈的基本操作
s.push(1);       // 入栈：将元素1压入栈顶
s.push(2);       // 入栈：将元素2压入栈顶

cout << s.top(); // 访问栈顶元素：输出2

s.pop();         // 出栈：移除栈顶元素(2)

cout << s.size();// 获取栈中元素个数：输出1
cout << s.empty();// 判断栈是否为空：输出false

// 3. 栈的遍历
while (!s.empty()) {
    cout << s.top() << " ";
    s.pop();
}

解题技巧

括号匹配问题

遇到左括号入栈，遇到右括号则判断是否与栈顶左括号匹配，匹配则出栈。

表达式求值

将十进制数不断除以目标进制，余数入栈，最后依次出栈得到转换结果。

单调栈应用

队列 (Queue)

队列的概念

队列是一种遵循"先进先出"(FIFO, First In First Out)原则的线性结构。 只能在队尾(rear)插入元素，在队头(front)删除元素，这是CSP初赛的重要考点。

队列的特点

  • 先进先出(FIFO)的数据结构
  • 在队尾插入元素，在队头删除元素
  • 插入操作称为入队(enqueue)
  • 删除操作称为出队(dequeue)
  • 可访问队头和队尾元素

C++队列的实现与操作

#include   // 包含队列头文件
using namespace std;

// 1. 队列的定义
queue q;

// 2. 队列的基本操作
q.push(1);       // 入队：将元素1加入队尾
q.push(2);       // 入队：将元素2加入队尾

cout << q.front(); // 访问队头元素：输出1
cout << q.back();  // 访问队尾元素：输出2

q.pop();         // 出队：移除队头元素(1)

cout << q.size(); // 获取队列中元素个数：输出1
cout << q.empty();// 判断队列是否为空：输出false

// 3. 队列的遍历
while (!q.empty()) {
    cout << q.front() << " ";
    q.pop();
}

数组例题：最大子数组和

题目描述

给定一个整数数组，找出其中连续子数组的最大和。例如，对于数组[-2,1,-3,4,-1,2,1,-5,4]， 连续子数组[4,-1,2,1]的和最大，为6。

解题思路

使用动态规划思想，遍历数组时维护两个变量：当前最大子数组和和全局最大子数组和。 对于每个元素，判断是加入当前子数组还是重新开始一个子数组。

代码实现

#include 
#include 
#include 
using namespace std;

int maxSubArray(vector& nums) {
    if (nums.empty()) return 0;
    
    int current_max = nums[0];
    int global_max = nums[0];
    
    for (int i = 1; i < nums.size(); i++) {
        // 要么将当前元素加入之前的子数组，要么从当前元素重新开始
        current_max = max(nums[i], current_max + nums[i]);
        // 更新全局最大值
        global_max = max(global_max, current_max);
    }
    
    return global_max;
}

int main() {
    vector nums = {-2, 1, -3, 4, -1, 2, 1, -5, 4};
    cout << "最大子数组和为：" << maxSubArray(nums) << endl;
    return 0;
}

字符串例题：最长回文子串

题目描述

给定一个字符串，找出其中最长的回文子串。例如，对于输入"babad"，输出可以是"bab"或"aba"； 对于输入"cbbd"，输出是"bb"。

解题思路

从每个字符向两边扩展，检查以该字符为中心的最长回文子串。 需要分别处理奇数长度和偶数长度的回文子串。

代码实现

#include 
#include 
using namespace std;

// 扩展函数：从中心向两边扩展寻找最长回文子串
int expandAroundCenter(string s, int left, int right) {
    while (left >= 0 && right < s.length() && s[left] == s[right]) {
        left--;
        right++;
    }
    return right - left - 1;  // 返回回文子串长度
}

string longestPalindrome(string s) {
    if (s.empty()) return "";
    
    int start = 0, end = 0;
    for (int i = 0; i < s.length(); i++) {
        int len1 = expandAroundCenter(s, i, i);      // 奇数长度
        int len2 = expandAroundCenter(s, i, i + 1);  // 偶数长度
        int len = max(len1, len2);
        
        if (len > end - start) {
            start = i - (len - 1) / 2;
            end = i + len / 2;
        }
    }
    
    return s.substr(start, end - start + 1);
}

int main() {
    string s = "babad";
    cout << "最长回文子串为：" << longestPalindrome(s) << endl;
    return 0;
}

栈例题：有效的括号

题目描述

给定一个只包括'('，')'，'{'，'}'，'['，']'的字符串，判断字符串是否有效。 有效字符串需满足：左括号必须用相同类型的右括号闭合；左括号必须以正确的顺序闭合。

解题思路

使用栈存储左括号，遇到右括号时检查是否与栈顶左括号匹配。若匹配则弹出栈顶元素， 否则字符串无效。最后检查栈是否为空，为空则有效。

代码实现

#include 
#include 
#include 
#include 
using namespace std;

bool isValid(string s) {
    stack st;
    // 创建括号匹配的映射表
    unordered_map mapping = {
        {')', '('},
        {'}', '{'},
        {']', '['}
    };
    
    for (char c : s) {
        // 如果是右括号
        if (mapping.find(c) != mapping.end()) {
            // 取栈顶元素，若栈为空则用一个特殊字符代替
            char top_element = st.empty() ? '#' : st.top();
            st.pop();
            
            // 检查是否匹配
            if (top_element != mapping[c]) {
                return false;
            }
        } 
        // 如果是左括号，入栈
        else {
            st.push(c);
        }
    }
    
    // 栈为空则所有括号都匹配
    return st.empty();
}

int main() {
    string s = "()[]{}";
    cout << (isValid(s) ? "有效" : "无效") << endl;
    return 0;
}

队列例题：滑动窗口最大值

题目描述

给定一个数组nums和滑动窗口的大小k，请找出所有滑动窗口里的最大值。 例如，对于nums = [1,3,-1,-3,5,3,6,7], k = 3，输出应为[3,3,5,5,6,7]。

解题思路

使用双端队列维护窗口中的最大值索引。队列中始终保持索引对应的元素递减， 新元素入队前移除所有比它小的元素。队首元素即为当前窗口的最大值。

代码实现

#include 
#include 
#include 
using namespace std;

vector maxSlidingWindow(vector& nums, int k) {
    vector result;
    deque dq;  // 存储索引，对应元素保持递减
    
    for (int i = 0; i < nums.size(); i++) {
        // 移除窗口外的元素
        if (!dq.empty() && dq.front() == i - k) {
            dq.pop_front();
        }
        
        // 移除所有小于当前元素的元素
        while (!dq.empty() && nums[dq.back()] < nums[i]) {
            dq.pop_back();
        }
        
        // 当前元素索引入队
        dq.push_back(i);
        
        // 当窗口大小达到k时，开始记录结果
        if (i >= k - 1) {
            result.push_back(nums[dq.front()]);
        }
    }
    
    return result;
}

int main() {
    vector nums = {1,3,-1,-3,5,3,6,7};
    int k = 3;
    vector res = maxSlidingWindow(nums, k);
    
    for (int num : res) {
        cout << num << " ";
    }
    // 输出：3 3 5 5 6 7
    return 0;
}
