数制转换

数制转换是CSP初赛的核心考点，包括二进制、八进制、十进制和十六进制之间的相互转换。掌握数制转换是信息学奥赛的基础技能， 本页面将系统讲解各类数制的表示方法及转换规则，并提供动态演示帮助理解。

数制基本概念

什么是数制？

数制也称计数制，是用一组固定的符号和统一的规则来表示数值的方法。在计算机科学中，常用的数制有：

二进制 (Binary)

基数为2，仅使用0和1两个数字，逢二进一。是计算机内部数据的表示形式。

示例: 1011₂

八进制 (Octal)

基数为8，使用0-7八个数字，逢八进一。常用于简化二进制表示。

示例: 13₈

十进制 (Decimal)

基数为10，使用0-9十个数字，逢十进一。是人类最常用的数制。

示例: 11₁₀

十六进制 (Hexadecimal)

基数为16，使用0-9和A-F(或a-f)，逢十六进一。常用于表示二进制数据。

示例: B₁₆

数制的位权表示法

任何数制的数都可以表示为各位数字与位权乘积之和。位权是基数的n次幂，n为数位的位置（从右向左，从0开始计数）。

通用公式

N = dₙ₋₁×rⁿ⁻¹ + dₙ₋₂×rⁿ⁻² + ... + d₁×r¹ + d₀×r⁰ + d₋₁×r⁻¹ + ...

其中：r为基数，dᵢ为第i位的数字，rⁱ为第i位的位权

二进制转十进制示例

1011₂ = 1×2³ + 0×2² + 1×2¹ + 1×2⁰

= 8 + 0 + 2 + 1 = 11₁₀

十六进制转十进制示例

A3₁₆ = 10×16¹ + 3×16⁰

= 160 + 3 = 163₁₀

数制转换方法

1. 其他进制转十进制

方法：按位权展开法，将每个数位上的数字乘以该位的位权，然后求和。

二进制 → 十进制

例：11010₂ 转换为十进制

= 1×2⁴ + 1×2³ + 0×2² + 1×2¹ + 0×2⁰
= 16 + 8 + 0 + 2 + 0
= 26₁₀

八进制 → 十进制

例：345₈ 转换为十进制

= 3×8² + 4×8¹ + 5×8⁰
= 3×64 + 4×8 + 5×1
= 192 + 32 + 5 = 229₁₀

十六进制 → 十进制

例：2A7₁₆ 转换为十进制

= 2×16² + 10×16¹ + 7×16⁰
= 2×256 + 10×16 + 7×1
= 512 + 160 + 7 = 679₁₀

2. 十进制转其他进制

方法：除基取余法，将十进制数除以目标进制的基数，取余数作为最低位， 然后将商继续除以基数，直到商为0，最后将所有余数倒序排列。

十进制 → 二进制

例：37₁₀ 转换为二进制

37 ÷ 2 = 18 余 1（最低位）
18 ÷ 2 = 9 余 0
9 ÷ 2 = 4 余 1
4 ÷ 2 = 2 余 0
2 ÷ 2 = 1 余 0
1 ÷ 2 = 0 余 1（最高位）
结果：100101₂

十进制 → 八进制

例：156₁₀ 转换为八进制

156 ÷ 8 = 19 余 4（最低位）
19 ÷ 8 = 2 余 3
2 ÷ 8 = 0 余 2（最高位）
结果：234₈

十进制 → 十六进制

例：249₁₀ 转换为十六进制

249 ÷ 16 = 15 余 9（最低位）
15 ÷ 16 = 0 余 15(F)（最高位）
结果：F9₁₆

3. 二进制与八/十六进制转换

利用2³=8和2⁴=16的关系，可以直接进行分组转换，无需经过十进制中间步骤。

二进制 ↔ 八进制

规则：每3位二进制数对应1位八进制数

二进制 → 八进制：

例：10110101₂ → 分组：010 110 101 → 2 6 5 → 265₈

八进制 → 二进制：

例：347₈ → 每位转3位：011 100 111 → 11100111₂

二进制 ↔ 十六进制

规则：每4位二进制数对应1位十六进制数

二进制 → 十六进制：

例：110101101₂ → 分组：0001 1010 1101 → 1 A D → 1AD₁₆

十六进制 → 二进制：

例：3F5₁₆ → 每位转4位：0011 1111 0101 → 1111110101₂

4. 八进制与十六进制转换

通常以二进制或十进制为中间桥梁进行转换：

八进制 → 十六进制

方法1：八进制 → 二进制 → 十六进制

例：257₈ → 010 101 111 → 10101111 → AF₁₆

方法2：八进制 → 十进制 → 十六进制

例：257₈ = 175₁₀ → AF₁₆

十六进制 → 八进制

方法1：十六进制 → 二进制 → 八进制

例：E3₁₆ → 1110 0011 → 111 000 11 → 703₈

方法2：十六进制 → 十进制 → 八进制

例：E3₁₆ = 227₁₀ → 343₈

数制转换解题技巧

快速转换技巧

1

二进制快速计算

记住2的幂次值：2⁰=1, 2¹=2, 2²=4, ..., 2¹⁰=1024，有助于快速计算二进制转十进制。

2

分组转换法

二进制与八/十六进制转换时，从右向左分组，不足补0，直接对应转换，无需经过十进制。

3

十六进制数字记忆

熟记A=10, B=11, C=12, D=13, E=14, F=15，避免转换时出错。

CSP初赛常见考点

1

基本转换计算

直接考查不同数制间的转换，如二进制转十进制、十六进制转二进制等。

2

数制运算

同一数制下的加减运算，或不同数制下数值大小比较。

3

计算机存储相关

结合字节(Byte)、位(bit)等概念考查，如计算存储容量对应的二进制位数。

常见错误及避免方法

常见错误

  • 分组转换时方向错误（应从右向左分组）
  • 除基取余时余数顺序颠倒（应将最后得到的余数作为最高位）
  • 混淆十六进制字母的数值（如将F误认为14）
  • 计算位权时指数错误（从0开始计数）
  • 转换过程中数字写错或漏写

避免方法

  • 转换过程分步书写，清晰记录每一步
  • 完成后反向验证，检查结果是否正确
  • 对复杂转换，可采用不同方法验证
  • 熟记常用数制对应表，提高转换速度
  • 注意数制标识，避免不同数制混淆

CSP初赛例题解析

例题1：基础转换

（2021年CSP初赛选择题）二进制数10110101对应的十进制数是多少？

解题步骤：

使用位权展开法：

10110101₂ = 1×2⁷ + 0×2⁶ + 1×2⁵ + 1×2⁴ + 0×2³ + 1×2² + 0×2¹ + 1×2⁰

= 128 + 0 + 32 + 16 + 0 + 4 + 0 + 1

= 181₁₀

答案：181

例题2：十六进制转换

（2020年CSP初赛选择题）十六进制数1A3F对应的二进制数是？

解题步骤：

将每位十六进制数转换为4位二进制数：

1 → 0001

A → 1010

3 → 0011

F → 1111

合并结果：1000101000111111₂

答案：1000101000111111

例题3：数制比较

（2019年CSP初赛选择题）下列数值中，最大的是？

A. 101101₂   B. 53₈   C. 45₁₀   D. 2D₁₆

解题步骤：

将所有选项转换为十进制比较：

A. 101101₂ = 1×32 + 0×16 + 1×8 + 1×4 + 0×2 + 1 = 45₁₀

B. 53₈ = 5×8 + 3 = 43₁₀

C. 45₁₀ = 45₁₀

D. 2D₁₆ = 2×16 + 13 = 45₁₀

比较结果：A=C=D>B，最大的是A、C、D

答案：ACD

例题4：综合应用

（2018年CSP初赛选择题）存储一个32×32点阵的汉字，需要占用多少字节？

解题步骤：

1. 计算总位数：32×32 = 1024位

2. 转换为字节（1字节=8位）：1024 ÷ 8 = 128字节

答案：128字节

答案：128
